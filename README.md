# Benchmarking on Tasks That Matter: Dataset Selection for Preserving Model Rankings

This repository contains the code accompanying the paper:

> **"Benchmarking on Tasks That Matter: Dataset Selection for Preserving Model Rankings"**

**Status**

This is **not** the final version of the code.  
A cleaned-up version will be uploaded later.

For now, the repository primarily contains exploratory and experimental code.

## Experiments

All experiments are implemented as Jupyter notebooks.

You can:

- inspect the experimental setup,
- reproduce individual results,
- and adapt the code to your own datasets and benchmarks

by running the notebooks in this repository.
